# TicTacToe

## How to Play
1. Two player game
2. Player 1 goes first, then player 2 goes
3. First to connect 3 in a row (horizontal, vertical, or diagnol)
4. Your name will shimmer when it is your turn

## Setup
1. Run `$ pod install`
2. Open `TicTacToe.xcworkspace`

## Launch
1. Choose simulator
2. Command + R

## Test
1. Command + U
